Coloca aquí tus imágenes del juego.

Sugerencias de nombres (puedes usar otros):
- background.png  -> Fondo del juego
- ball.png        -> Textura para las bolas automáticas

Cómo se usan:
- En el juego, usa los botones "Cargar Fondo" y "Cargar Textura Bola" del panel lateral para seleccionar los archivos desde esta carpeta.
- Si no hay imagen cargada, el juego dibuja el fondo blanco y las bolas como círculos de colores.

Nota: Esta carpeta no es código Java. Es sólo para recursos (imágenes).
