# Contributing

Gracias por interesarte en contribuir a Juego_Bolas. Este proyecto es educativo y cualquier mejora es bienvenida.

Reglas básicas:
- Usa la rama `main` para el código estable.
- Haz un fork y un pull request para cambios grandes.
- Incluye una descripción corta de lo que cambias y por qué.

Formato de código:
- Java 11.
- Mantén los comentarios claros y concisos.

Reportar bugs:
- Abre un issue con pasos para reproducir y el error esperado/real.
